## Git Workflow
- Edit on WSL → `git push`
- Pull on Windows → `cargo tauri dev`
- Never copy files manually

## Common Issues
1. Ubuntu detection fails → `wsl.exe -d Ubuntu -- echo OK` check
2. Tauri config → no `title` or `env` in `app` section
3. Shell plugin → only `open: true`, no `scope`
4. `tungstenite::connect` → use `&str` not `Url`
5. `Cow<str>` → `.to_string()` for String fields
